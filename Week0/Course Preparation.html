Course Preparation

This course is a condensed introduction to JavaScript programming, and the course is very demanding. You will need to keep up with the work that is assigned each week, and no late work will be accepted. 

When you enrolled in the course, there was a warning that completion of CPSC 1030 was strongly recommended prior to enrolling in this course. If you have not taken CPSC 1030, or if you need a refresher in HTML and CSS, please do the the starter activities in this module to prepare yourself.
