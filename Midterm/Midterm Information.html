CPSC 1045 Midterm Exam - Please read all of the following instructions. 

  • The exam will be available to start any time after noon PDT on Friday, July 17th but before noon PDT on Saturday, July 18th. The exam is a 2 hour timed exam. Once you start the exam, you have 120 minutes then the exam will submit whether you are finished or not. 
  • Will be delivered via D2L quiz, but you must also submit a zip file of your code to the exam dropbox within 5 minutes of completing the exam. If you do not submit your code to the dropbox, you may receive 0 on the programming portion of the exam. 
  • The exam will cover everything up to and including loops and events (weeks 1-3 inclusive). The exam will consist of about 5 short answer questions and two small-ish programming tasks
  • The short answer questions may ask you to determine the input/output of the code or may be similar to the kinds of questions you see on the quizzes.
  • The programming tasks will ask you to write the JavaScript functions to perform a task. This may include getting input (textbox or prompt), giving output (console, alert, write to webpage), and will definitely include if/else, loops, and events (click and key). 
  • The exam is partially open book. You can use any course resources including notes, labs, assignments, D2L, and the textbook. However you are not allowed to work with any other person. This means no instant messaging, screen sharing, social media, communicating, or other collaborative tools.
